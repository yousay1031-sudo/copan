# 児童デイサービス コパン（COPAN）ホームページ

![コパン ロゴ](https://www.genspark.ai/api/files/s/xYU9kYIx)

北海道帯広市の放課後等デイサービス「コパン（COPAN）」の公式ホームページです。

---

## 🎨 デザインコンセプト

**温かみのあるオレンジトーンの優しいデザイン**

- **オレンジベースのパステルカラー配色**: メインオレンジ、ピーチ、グリーン、ピンク、ブルーの柔らかい色合い
- **2枚重ねの写真レイアウト**: 
  - 大小2枚の写真を重ねた立体的なデザイン
  - 絵の具風のにじみ枠で温かい雰囲気
  - 右下に絵の具風の装飾ブロブ（グリーン＆オレンジ）
  - 周りに葉っぱの装飾が浮遊
- **🎨 不規則な横長楕円形の画像フレーム**:
  - **絵の具で描いたような柔らかいにじみ枠**
  - グラデーションを使った滑らかで自然な境界
  - 各セクションで異なる色合いの枠（オレンジ、ライトオレンジ、グリーン）
  - 有機的で手作り感のある温かいデザイン
- **かわいい装飾**: 
  - 🍃 **葉っぱ**: ヒーロー・プログラムセクション
  - ⭐ **星**: コパンについて・フッターセクション
  - 💕 **ハート**: プログラムセクション
  - ☁️ **雲**: 1日の流れセクション
  - 🌸 **花**: ご利用案内セクション
  - 🦋 **蝶々**: お問い合わせセクション
  - 🌈 **虹**: お問い合わせセクション
  - 🌳 **木**: アクセスセクション
  - ☀️ **太陽**: アクセスセクション
- **アニメーション**: すべての装飾が優しく動き、生き生きとした印象
- **AI生成画像**: 子どもたちの活動シーンを6枚配置

## 📝 Google Maps の埋め込み方法

3. 埋め込みコードをコピー
4. `index.html` の `.map-placeholder` 部分を以下のように置き換え：

```html
<div class="access-map">
    <iframe 
        src="埋め込みURL" 
        width="100%" 
        height="450" 
        style="border:0; border-radius: 30px;" 
        allowfullscreen="" 
        loading="lazy" 
        referrerpolicy="no-referrer-when-downgrade">
    </iframe>
</div>
```
